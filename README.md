# DevTask – React + Vite

This project is built with React + Vite and styled using Tailwind CSS.  
It includes recruiter‑ready flows:

- **Welcome Screen** with animated numbered bubbles
- **Create Account Screen** with regex validation, password strength meter, and show/hide toggle
- **Login Screen** for registered users
- **Account Settings Screen** with profile image upload/change/remove and confirmation modal

---

## 🚀 Features
- Pixel‑perfect UI built with React + Vite
- Smooth bubble animations (top, down, left, right, diagonal)
- Real‑time regex validation for all form fields
- Password strength meter + show/hide toggle
- Modular component architecture with React Router navigation
